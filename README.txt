# Diego Moche Domicilios — Términos y Condiciones

Este paquete contiene un sitio web estático listo para publicar.

## Contenido
- `index.html` — página principal.
- `assets/logo-diego-moche.png` y `assets/logo-hojas.png` — logos.
- Enlaces a Instagram y Facebook ya configurados.

## Cómo publicarlo
1) **Netlify (más fácil)**
   - Entra a https://app.netlify.com/drop
   - Arrastra la carpeta `diego-moche-terminos-site` o el ZIP.
   - Netlify te da una URL pública al instante (puedes personalizar el dominio).

2) **GitHub Pages**
   - Crea un repo y sube la carpeta.
   - En Settings → Pages, selecciona `main` y `/root`.
   - Tu página quedará disponible en minutos.

3) **Cualquier hosting**
   - Sube todo el contenido del ZIP al public_html o carpeta web del hosting.
